== Description ==

A quick modification of the ubiquitious Hello Dolly plugin, to present quotes from
Leonardo Da Vinci in the admin section.

== Installation ==

This section describes how to install the plugin and get it working.

1. Disable the Hello Dolly plugin in your WordPress Admin > Plugins panel.

2. Upload the hello-davinci folder to your plugins directory via FTP, or install from the plgins admin panel.

3. Activate the the plugin in your admin panel.

4. Yep, that’s it, you’re done. Behold the wisdom!

== Frequently Asked Questions ==

= What about the Hello Dolly Plugin? =

You will need to deactivate the Hello Dolly plugin in your admin panel.
